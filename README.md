#  RClick

Config you MacOS ContextMenu items, useing Latest Swift and SwiftUI. 



## Feature

- [x] use someApp open path
- [x] copy path
- [x] delete dir or file
- [x] create file like txt json markdown docx pptx xlsx etc.
